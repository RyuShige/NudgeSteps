# NudgeSteps

今日のこれまでの歩数（緑）と昨日のこれまでの歩数（オレンジ）を直感的に比較可能にすることで、ユーザの歩数促進を図ります。

薄いオレンジは昨日の合計歩数、薄い緑は昨日の合計歩数＋ー500を表示してます。
これは、比較する歩数が＋ー500以内であれば、ユーザの歩数が有意に促進されるという研究結果に基づいています。https://dl.acm.org/doi/10.1145/2971648.2971754
